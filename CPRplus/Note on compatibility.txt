CPRplus is compatible with overhaul mods listed below, but you must make sure to provide a compatibility measure.


Patch for CK2 Plus: https://forum.paradoxplaza.com/forum/index.php?threads/ck2plus-cprplus-compatibility-patch.1025356/

Patch for A Game of Thrones mod (you must run the game with both CPRplus the submod enabled): http://steamcommunity.com/sharedfiles/filedetails/?id=942303847

Patch for Seven Kingdoms (you must run the game with both CPRplus the submod enabled): https://steamcommunity.com/sharedfiles/filedetails/?id=1360765985

Historical Immersion Project (install CPRplus from the HIP installer): https://forum.paradoxplaza.com/forum/index.php?threads/historical-immersion-project-download-faq.748775/


CPRplus is not compatible with overhaul and portrait mods out of the box.

Feel free to post in the CPRplus thread in the Paradox forums for help!
(https://forum.paradoxplaza.com/forum/index.php?threads/mod-cprplus.884666/)
